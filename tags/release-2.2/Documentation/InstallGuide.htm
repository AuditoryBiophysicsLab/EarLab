Desktop EarLab Installation Guide

 

System Requirements

  • Pentium III or better
  • Windows 2000 or XP
  • Memory: 256 MB minimum, 512 MB  or more recommended

 

Overview

EarLab is a virtual laboratory that is intended to allow scientists from a wide range of backgrounds to perform experiments on large-scale computational models of the mammalian auditory pathways. The EarLab simulations are based on interchangeable building blocks or modules each of which represents a different component of the auditory system or sound-source, data-acquisition and data-visualization systems. The modules are designed to run in a distributed heterogeneous computing environment and to be configured at run time. The physiological modules are designed to be species independent with species dependent parameters loaded at run time.  The current system includes modules that represent sound sources, propagation between the source and the tympanic membrane, and the subthalamic auditory pathways. These modules can all run on a single CPU (Desktop EarLab) or they can be distributed across many heterogeneous CPUs connected by a local or a wide-area network (Distributed EarLab).

EarLab takes time and divides it up into segments called frames. A frame may continue many one sample or many samples depending on the relative size of the frame and the inter-sample interval.  The modules compute independently during each frame and then exchange information at the end of the frame.

In this document we will use the term model to mean a group of modules and their interconnections.  EarLab experiments consist of multiple runs where a run is a single simulation on a specific model with a single set of parameters. A model consists of a choice of modules and their interconnections. A model diagram file is used to specify the modules to be used, the dimensionality of the module outputs, and the source of module inputs. The module parameters are specified in a parameter file. 

The Desktop EarLab installation package includes the EarLab ExperimentManager, the EarLab Simulator, a library of modules, a library of sound files, data browsing software (EarLab DataViewer), documentation and help files, and a collection of preconfigured models.

Installation

To install Desktop EarLab, insert the EarLab CD into your CD drive and the installer should launch automatically. If for some reason the installer does not launch, open the CD and run DesktopEarLabSetup.exe.  When the installer launches properly, you will see the following screen:

Click Next to continue

If you agree to the terms of the software license, indicate that by clicking on the I accept this agreement radio button, and clicking Next, as seen here

Again, click Next, to accept the default installation folder

Again, click Next, to perform a full installation

Again, click Next, to accept the default program group in the Start menu

Again, click Next, to allow the installer to place an icon on your desktop

Click Install, to allow the installer to copy the necessary files to your computer

After the installation completes, click Finish and the installer will launch the EarLab Experiment Manager and display instructions for its use
