CaiIC21.0.0.09/18/2008Anderson DJ
 
Classification:
Structure Substructure Cell
inferior colliculus [IC], colliculus caudalis, posterior colliculus central nucleus of inferior colliculus [CIC], Nucleus centralis, chief nucleus of inferior colliculus
 
Abstract:
The CaiIC2 module implements a model similar to the Cai et al. (1998) model for neurons of the central nucleus of the inferior colliculus (ICC). While there is wide variation in membrane characteristics between the neurons of this nucleus, and across their population they receive inputs from a number of sources, the CaiIC2 module focuses primarily on the subpopulation of ICC neurons that are sensitive to interaural time delay (ITD) and respond best to low-frequency auditory inputs. The input structure of the CaiIC2 module is the same as that of the CaiIC module, and the conductance for additive long-lasting adaptation of Cai et al. (1998) is included. However the single-compartment Hodgkin-Huxley model of the CaiIC2 module is otherwise that of Rothman and Manis (2003c), including kinetics based on their measurements in the ventral cochlear nucleus of three K+ channels specific for IKLT, IKHT, and IKA (Rothman and Manis 2003a), and additional studies describing the kinetics of INa and Ih elsewhere in the mammalian nervous system. As in the Cai et al. model (1998b) the compartment incorporates an additional conductance for additive long-lasting adaptation following each action potential. The module’s ResponseType parameter is used to select from the Rothman and Manis parameters sets for response types I-c, I-t, I-II, II-I and II, which are similar to response types in ICC neurons (Sivaramakrishnan and Oliver, 2001). Although adaption characteristics and response type are selected through independent parameters, in ICC neurons the K+ current controlling adaptation is co-expressed with the basic ion-channel currents that support Type I responses, and not with the additional currents that support Type II responses.
Description:
The CaiIC2 module implements an array of ICC neurons with two dimensions of lengths that are specified in the runtime gui or diagram file. The length of the first dimension sets the number of frequency channels, and the length of the second dimension sets the number of model cells per frequency channel equal to the number of characteristic delays in each of two MSO modules acting as an input. In order to implement the Cai et al. model (1998), the CaiIC2 module receives one excitatory input stream driven by an ipsilateral MSO module, and one inhibitory input stream driven by a contralateral MSO module. Each model ICC neuron receives one excitatory synaptic input, and one inhibitory synaptic input. While this strict adherence to the original model may be restrictive, the CaiIC2 module thus incorporates the original model’s convergent inputs of same and different characteristic delays. The inputs, individual cells within the BrugheraMSO module, are organized by frequency channel and characteristic delay. The number of frequency channels is equal between ipsilateral and contralateral MSO modules, as are the number of characteristic delays. Within a frequency channel in the MSO modules, the characteristic delays progress from zero to the maximum value in equal increments. The CaiIC2 module has the same number of frequency channels as each MSO module. Within a frequency channel the CaiIC2 module has a number of neurons equal to the number of characteristic delays in each MSO module, and within that frequency channel each contralateral model MSO cell (with its different characteristic delay) projects an inhibitory input to a different model IC neuron. Setting the ExcitatoryMSOLagNumber parameter (to a value between zero and the number of characteristic delays minus one) selects exactly one ipsilateral model MSO cell of that particular characteristic delay so that it projects an excitatory input to each of the model IC neurons within that frequency channel. In this way the effects of similarity and differences in the characteristic delays between the excitatory input and inhibitory input can be examined.

Parameters:
Name Type Description
SampleRate_Hz float The sample rate, in hertz, sets the effective time resolution of the input and output streams, and the output time resolution of the Runge Kutta differential equation solver.
FrameLength_Seconds float duration of a processing block
ExSynapticStrength float ExSynapticStrength, Ge, given the unit nanosiemens (nS), sets the maximum value of the increment in excitatory conductance associated with an excitatory synaptic input firing. The time course of the conductance increment is a scaled alpha function, t(Ge/Tau)exp(1 - t/Tau), with the time constant of decay, Tau, specified by ExSynapticTau_Seconds.
ExSynapticTau_Seconds float time constant of the excitatory synaptic conductance
InhSynapticStrength float InhSynapticStrength, Gi, given the unit nanosiemens (nS), sets the maximum value of the increment in inhibitory conductance associated with an inhibitory synaptic input firing. The time course of the conductance increment is a scaled alpha function, t(Gi/Tau)exp(1 - t/Tau), with the time constant of decay, Tau, specified by InhSynapticTau_Seconds.
InhSynapticTau_Seconds float time constant of the inhibitory synaptic conductance
CellTemperature_DegreesC float The cell temperature (in degrees Celsius) is typically a mammalian body temperature, such as 38, for simulation of in vivo experiments, and 22 for in vitro.
AHPIncrement_Siemens float AHPIncrement_Siemens is an optional parameter, added to the Rothman and Manis 2003c compartment, with a default value of zero. When set to a non-zero value, AHPIncrement_Siemens in combination with TauAHP_Seconds implements adaptation by means of a potassium conductance with a step increase and a first-order exponential decay. Exactly 0.5 ms after each action potential, the conductance is incremented by AHPIncrement_Siemens, and then decays with time constant TauAHP_Seconds.
InhInputDelay_Seconds float The delay in seconds applied to the inhibitory input.
ExcitatoryMSOLagNumber float Setting the ExcitatoryMSOLagNumber parameter (to a value between zero and the number of characteristic delays minus one) selects exactly one ipsilateral model MSO cell of that particular characteristic delay so that it projects an excitatory input to each of the model IC neurons within that frequency channel.
ResponseType int An optional parameter, with a default value of 2 (Type II). TheBushyCell2 module can be set for a response type associated with bushy cells or other VCN neurons. Valid values of ResponseType include and implement the response types in Rothman and Manis (2003c): RM2003 Type I-c: ResponseType = 1; RM2003 Type I-t: ResponseType = 11; RM2003 Type I-II: ResponseType = 12; RM2003 Type II-I: ResponseType = 21; RM2003 Type II: ResponseType = 2.
VmemFilename bool An optional parameter, which writes the membrane voltage in milliVolts to the specified file.
geFilename bool An optional parameter, which writes the excitatory synaptic conductance (unit nS) to the specified file.
giFilename string An optional parameter, which writes the inhibitory synaptic conductance (unit nS) to the specified file.
gNaFilename bool An optional parameter, which writes the sodium conductance (unit nS) to the specified file.
gKAFilename string An optional parameter, which writes the transient A-type potassium conductance (unit nS) to the specified file.
gKHTFilename string An optional parameter, which writes the high-threshold potassium conductance (unit nS) to the specified file.
ghFilename string An optional parameter, which writes the hyperpolarization-activated cation conductance (unit nS) to the specified file.
gAHPFilename string An optional parameter, which writes the optional adaptation conductance (unit nS) to the specified file.
TauAHP_Seconds float TauAHP_Seconds is an optional parameter, added to the Rothman and Manis 2003c compartment, with a default value of (1.0/SampleRate_Hz). When parameter AHPIncrement_Siemens is set to a non-zero value, these two parameters implement adaptation by means of a potassium conductance with a step increase and a first-order exponential decay. Exactly 0.5 ms after each action potential, the conductance is incremented by AHPIncrement_Siemens, and then decays with time constant TauAHP_Seconds.
