VoigtDCNCell1.0.0.09/23/2008Anderson DJ
 
Classification:
Structure Substructure Cell
cochlear nuclei [Co], nuclei cochleares, cochlear nucleus dorsal cochlear nucleus [Dco], Nucleus cochlearis dorsalis, dorsal division of cochlear nucleus fusiform cell
 
Abstract:
The VoigtDCNCell module implements sets of model dorsal cochlear nucleus (DCN) neurons as in the Zheng and Voigt (2006) model for response maps of the DCN. Each individual model neuron is an electric circuit model based on the MacGregor (1987) neuromime, with parallel branches containing membrane capacitance, leakage conductance, a potassium channel, and synaptic inputs. Voltage sources equal to the reversal potentials are included in series with the potassium and synaptic conductances. Each excitatory or inhibitory input adds another parallel branch to the circuit. Three types of DCN neurons are included in the model: Pyramidal (fusiform) or P-cells representing principal cells, I2-cells representing inhibitory interneurons with type-II response maps, and W-cells representing wideband inhibitors which receive a wide frequency range of auditory nerve (AN) -fiber inputs. AN fibers excite the W, I2, and P cells. W-cells inhibit P-cells and I2-cells. I2-cells inhibit P-cells. Typical intrinsic and connection parameters are given in Zheng and Voigt (2006). The connectivity for the VoigtDCNCell modules is managed by the SpikeStreamMapper SupportClass, using files containing connectivity maps generated in Matlab.

Parameters:
Name Type Description
SampleRate_Hz float The sample rate, in hertz, sets the effective time resolution of the input and output streams, and the output time resolution of the Runge Kutta differential equation solver.
FrameLength_Seconds float duration of a processing block
Tau_m_Seconds float The membrane time constant in seconds
b_K float When the membrane reaches voltage threshold (V_th_Volts), the time derivative of the potassium conductance is incremented by b_k divided by Tau_K_Seconds.
V_th_Volts float The membrane voltage threshold for registering an action potential. Note that the resting potential of the model VoigtDCNCell is zero.
Tau_K_Seconds float The time constant of the potassium conductance in seconds.
E_K_Volts float The reversal potential, in Volts, of the potassium current
E_ex_Volts float The reversal potential, in Volts, of the excitatory synapse
E_in_Volts float The reversal potential, in Volts, of the inhibitory synapses
EPSP_Strength float The increment in excitatory synaptic conductance that occurs with each input event. Note that conductances in this module are dimensionless because they are divided by G, the leakage conductance, a.k.a. the resting conductance.
IPSP_Strength1 float The increment in the first inhibitory synaptic conductance that occurs with each input event.
IPSP_Strength2 float The increment in the second inhibitory synaptic conductance that occurs with each input event.
ExcitatorySynapticTau_Seconds float The time constant, in seconds, of the excitatory synaptic conductance
InhibitorySynapticTau_Seconds1 float The time constant, in seconds, of the first inhibitory synapse
InhibitorySynapticTau_Seconds2 float The time constant, in seconds, of the second inhibitory synapse
