Bushy Cell (Rothman93)1.0.0.08/3/2005Anderson DJ
 
Classification:
Structure Substructure Cell
cochlear nuclei [Co], nuclei cochleares, cochlear nucleus anterior division of the anteroventral cochlear nucleus [AVCN-A] spherical bushy cell
cochlear nuclei [Co], nuclei cochleares, cochlear nucleus anterior division of the anteroventral cochlear nucleus [AVCN-A] globular bushy cell
 
Abstract:
The BushyCell module simulates individual bushy cells of the ventral cochlear nucleus (VCN). Each instance of BushyCell incorporates a single-compartment Hodgkin-Huxley neuron model that simulates the membrane activity and discharge patterns of a bushy cell. The neural compartment is defined by an electric circuit equation incorporating the membrane currents associated with capacitance, leakage, ion channels, and synaptic inputs. The characteristics of the three voltage-sensitive ion-channel conductances (inactivating sodium, delayed-rectifier high-threshold potassium, and low-threshold potassium) were derived from voltage-clamp data taken from the VCN (Manis and Marx, 1991). The low-threshold potassium channel plays a key role in coincidence detection and synchronization to low-frequency stimuli, by shortening the duration of excitatory post synaptic potentials so that subthreshold inputs must be highly coincident to sum effectively and produce an action potential. Closely based on the model of Rothman et al. (1993), the default compartment of the BushyCell module is that of Brughera et al. (1996), including its modifications that smooth and slow the ion-channel activity slightly and increase the overall potassium conductance near resting potential. The original Rothman et al. 1993 compartment can be selected using the CompartmentClassName parameter. Steady-state inhibition, used in the original Rothman 1993 model as a mechanism to increase the variation of inter-spike intervals (ISIs) in bushy cells of high characteristic frequency, has been omitted here because regularity in ISIs appears less of a problem with the Brughera et al. 1996 compartment.
Description:
The BushyCell module simulation implements an array of model VCN neurons with two dimensions of lengths that are specified in the runtime gui or diagram file. The length of the first dimension sets the number of frequency channels, and the length of the second dimension sets the number of model cells per frequency channel. The stream of output discharge times from the BushyCell module is commonly used as input to an MSO module. The BushyCell typically receives synaptic inputs from spiking model auditory nerve fibers, such as from module ANSpike, with an equal number of frequency channels. The number of AN fibers per frequency channel must be greater than or equal to the MaxSynapseCount parameter, which sets the number of synaptic inputs to each model VCN cell. The number of model VCN cells per frequency channel is arbitrary, but given that all are applied to the input of a subsequent model MSO cell, values in the range of 8-20 are typical. A ResponseType parameter has been added to set the maximum conductances of the ion-channels for Type I or Type II responses to current clamp. Type II is the default, and is associated with bushy cells; Type I with stellate cells. In a Type II model cell, manipulation of the number, strength, and CF of the synaptic inputs, can produce a variety of PSTHs associated with spherical and globular bushy cells (Rothman et al. 1993). At present, the connectivity from the ANSpike module remains within a single-CF, and the SynapticStrength parameter sets an equal strength for all synapses. The BushyCell module uses the neural compartment model of Brughera et al. (1996), which is a slightly modified version of that in the Rothman et al. (1993) model for bushy cells of the VCN. The Brughera96 compartment is identical to the Rothman93 compartment with three exceptions: 1) The Brughera96 compartment has a higher leakage reversal potential of 22 mV (vs. 2.8 mV in Rothman93) 2) The Brughera96 compartment has slightly lower temperature factors: TF(2.6655) in place of TF(3) TF(5.6569) in place of TF(10) 3) The Brughera96 compartment has reduced two large, abrupt variations in voltage-sensitive rate parameters present in Rothman93. The third modification gives the Brughera96 compartment smaller, more gradual increases in the rate parameters, beta_n and alpha_h, near and below resting membrane potential. The second terms in the expressions for beta_n and alpha_h are reduced by a factor of 1000 (see Rothman et al. 1993, Appendix). In addition the temperature factor was eliminated in the second term for beta_n, further reducing this term by a factor of 4.8 at temperature 38C. The first terms in the expressions for beta_n and alpha_h are unmodified, except for the difference in temperature factors. Collectively, these modifications slow the model and increase the overall potassium conductance near resting potential. As a result, coincidence detection ability is increased slightly. In model MSO cells, the more gradual recovery from refractory effects provides a more realistic modulation of output synchronization in response to ITD, because bilaterally out-of-phase inputs are less likely to cause responses primarily to ipsilateral inputs or primarily to contralateral inputs.

Parameters:
Name Type Description
SampleRate_Hz double Sample Rate, in Hertz, of the input file
FrameLength_Seconds float duration of a processing block (sec)
SynapticStrength float SynapticStrength in nanosiemens (nS) sets the maximum value of the increment in excitatory conductance associated with an excitatory synaptic input firing. The time course of the conductance increment is a scaled alpha function (t*exp(-t/Tau)) with time constant SynapticTau_Seconds.
SynapticTau_Seconds float The time constant of decay (sec) for synaptic conductance. See SynapticStrength for further explanation.
CellTemperature_DegreesC float The temperature (in degrees Celsius) within the cell. Typically 38 for simulation of in vivo experiments, and 22 for in vitro.
MaxSynapseCount int The number of synaptic inputs to the cell.
ResponseType int An optional parameter, with a default value of 2 (Type II). Type I: ResponseType = 1; Type II: ResponseType = 2.
geFilename string An optional parameter, which writes the excitatory synaptic conductance (unit nS) to the specified file.
giFilename string An optional parameter, which writes the inhibitory synaptic conductance (unit nS) to the specified file.
gNaFilename string An optional parameter, which writes the sodium conductance (unit nS) to the specified file.
gKHTFilename string An optional parameter, which writes the high-threshold potassium conductance (unit nS) to the specified file.
gKLTFilename string An optional parameter, which writes the low-threshold potassium conductance (unit nS) to the specified file.
gAHPFilename string An optional parameter, which writes the optional adaptation conductance (unit nS) to the specified file.
CompartmentClassName string An optional parameter, with a default value of "Brughera96Compartment", and optional value of "Rothman93Compartment".
