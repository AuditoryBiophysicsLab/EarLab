BushyCell2 (RothmanManis2003)1.0.0.09/5/2008Anderson DJ
 
Classification:
Structure Substructure Cell
cochlear nuclei [Co], nuclei cochleares, cochlear nucleus
 
Abstract:
The BushyCell2 module simulates individual neurons of the ventral cochlear nucleus (VCN). Each instance of BushyCell2 incorporates a single-compartment Hodgkin-Huxley neuron model that simulates the membrane activity and discharge patterns of a VCN neuron (Rothman and Manis 2003c). In VCN neurons, Rothman and Manis (2003a) found a differential expression of three distinct potassium currents, and stressed that the specific compliment of K+ currents is not only important in determining responses to injected currents and acoustic stimuli, but also in determining action potential shape, rate, adaptation, and regularity. Rothman and Manis (2003b) conducted a detailed analysis quantifying the kinetics of the activation and (when present) inactivation of the three K+ currents: the transient A-type (IKA), the dendrotoxin-sensitive low-threshold IKLT, and the high-threshold rectifying IKHT. In addition to these three currents, the model incorporates membrane capacitance and expressions for sodium current INa, leakage current ILEAK, the hyperolarization-activated cation current Ih. The BushyCell2 module can simulate either bushy cells which project to the superior olivary complex or medial nucleus of the trapezoid body, or stellate cells which project to the inferior colliculus or dorsal cochlear nucleus. The module’s ResponseType parameter selects Rothman and Manis’s parameters for responses of Type I-c, I-t, I-II, II-I, or II.
Description:
Each simulation implements an array of model VCN neurons with two dimensions with lengths that are specified in the runtime gui or diagram file. The length of the first dimension sets the number of frequency channels, and the length of the second dimension sets the number of model cells per frequency channel. Typically the BushyCell2 module is driven by an spiking auditory nerve (AN) model, such as ANSpike, with an equal number of frequency channels. The number of AN fibers per frequency channel must be greater than or equal to the MaxSynapseCount parameter, which sets the number of synaptic inputs to each model VCN cell. The number of model VCN cells per frequency channel is arbitrary, but given that all are applied to the input of a subsequent model MSO cell, values in the range of 8-20 are typical.

Parameters:
Name Type Description
SampleRate_Hz float Sample Rate, in Hertz, of the input file
FrameLength_Seconds float duration of a processing block
SynapticStrength float SynapticStrength in nanosiemens (nS) sets the maximum value of the increment in excitatory conductance associated with an excitatory synaptic input firing.
CellTemperature_DegreesC float The temperature (in degrees Celsius) within the cell, typically mammalian body temperature, 38, for simulation of in vivo experiments, and 22 for in vitro.
MaxSynapseCount int The number of synaptic inputs to the cell.
ResponseType int An optional parameter, with a default value of 2 (Type II). TheBushyCell2 module can be set for a response type associated with bushy cells or other VCN neurons. Valid values of ResponseType include and implement the response types in Rothman and Manis (2003c): RM2003 Type I-c: ResponseType = 1; RM2003 Type I-t: ResponseType = 11; RM2003 Type I-II: ResponseType = 12; RM2003 Type II-I: ResponseType = 21; RM2003 Type II: ResponseType = 2.
AHPIncrement_Siemens float AHPIncrement_Siemens is an optional parameter, added to the Rothman and Manis 2003c compartment, with a default value of zero. When set to a non-zero value, AHPIncrement_Siemens in combination with TauAHP_Seconds implements adaptation by means of a potassium conductance with a step increase and a first-order exponential decay. Exactly 0.5 ms after each action potential, the conductance is incremented by AHPIncrement_Siemens, and then decays with time constant TauAHP_Seconds.
VmemFilename string An optional parameter, which writes the membrane voltage in milliVolts to the specified file.
geFilename string An optional parameter, which writes the excitatory synaptic conductance (unit nS) to the specified file.
giFilename string An optional parameter, which writes the inhibitory synaptic conductance (unit nS) to the specified file.
gNaFilename string An optional parameter, which writes the sodium conductance (unit nS) to the specified file.
gKAFilename string An optional parameter, which writes the transient A-type potassium conductance (unit nS) to the specified file.
gKLTFilename string An optional parameter, which writes the low-threshold potassium conductance (unit nS) to the specified file.
gKHTFilename string An optional parameter, which writes the high-threshold potassium conductance (unit nS) to the specified file.
ghFilename bool An optional parameter, which writes the hyperpolarization-activated cation conductance (unit nS) to the specified file.
gAHPFilename string An optional parameter, which writes the optional adaptation conductance (unit nS) to the specified file.
SynapticTau_Seconds float The time constant of the conductance synaptic conductance
TauAHP_Seconds float TauAHP_Seconds is an optional parameter, added to the Rothman and Manis 2003c compartment, with a default value of (1.0/SampleRate_Hz). When parameter AHPIncrement_Siemens is set to a non-zero value, these two parameters implement adaptation by means of a potassium conductance with a step increase and a first-order exponential decay. Exactly 0.5 ms after each action potential, the conductance is incremented by AHPIncrement_Siemens, and then decays with time constant TauAHP_Seconds.
