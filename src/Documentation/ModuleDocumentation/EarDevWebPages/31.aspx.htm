BrugheraMSO21.0.0.09/12/2008Anderson DJ
 
Classification:
Structure Substructure Cell
superior olivary complex [Sol], regio olivaris superioris, superior olivary nucleus medial superior olivary nucleus [MSO], medial superior olive, chief superior olivary nucleus bipolar cell
 
Abstract:
The BrugheraMSO2 module simulates individual principal cells of the medial superior olive (MSO), which receive bilateral excitatory inputs from the left and right cochlear nuclei, and have responses sensitive to the interaural time delay (ITD) of low-frequency auditory stimuli. The module also incorporates bilateral inhibitory inputs, representing the contralateral and ipsilateral inhibitory inputs from the medial and lateral nuclei of the trapezoid body (MNTB and LNTB), respectively. Each instance of the module employs a single-compartment Hodgkin-Huxley neuron model derived for Type II ventral cochlear nucleus neurons, including expressions for IKLT, IKHT, INa, ILEAK, and Ih (Rothman and Manis 2003c) with characteristics that are likely similar to those in principal MSO cells. IKLT and INa channels have been shown to enhance coincidence detection in MSO cells, and Ih channels depolarize the resting membrane sufficiently to partially inactivate INa and activate IKLT (Svirskis et al. 2004; Scott et al. 2005). The activation of IKHT and IKLT contributes to membrane repolarization during an action potential, and the activation of the low-threshold potassium conductance (gKLT) produces faster membrane time constants near resting potential, reducing the duration of EPSPs. The module can be run using bilateral excitatory and bilateral inhibitory inputs (EEII), or simply with bilateral excitatory inputs (EE). Additional configurations, such as a model cell with ipsilateral excitatory inputs and contralateral inhibitory inputs (EI), can be rendered using the EEII mode and applying the active inputs along with inactive inputs (without spikes).
Description:
Each simulation implements an array of model MSO neurons with two dimensions of lengths that must be specified in the runtime gui or diagram file. The first dimension is the number of frequency channels, and the second dimension is the number of internal characteristic delays (Lags) per frequency channel. The parameter MaxLag_Seconds sets the maximum characteristic delay. During a simulation, a different characteristic delay, ranging in equal increments from zero to the value of MaxLag_Seconds, is applied to each model MSO cell along the characteristic-delay dimension. ITD may be introduced in the peripheral stimulus waveforms, or alternatively, if the MSO is the final processing module, the characteristic delay of all the model MSO neurons may be considered zero, and the applied internal delays may be considered as positive external ITDs. In this alternative case, the effect of negative ITDs may be simulated by reversing the ipsilateral and contralateral inputs. Typically the BrugheraMSO2 module is driven by ipsilateral and contralateral BushyCell2 modules, or for simplicity, by ipsilateral and contralateral ANSpike modules. The number of frequency channels in the ipsilateral and contralateral input modules must equal that of the BrugheraMSO2 module. If only two inputs are specified, they are implemented as excitatory. With four inputs, the first two are implemented as excitatory, and the last two as inhibitory. The first and third inputs are considered the ipsilateral inputs, and the second and fourth inputs the contralateral.

Parameters:
Name Type Description
FrameLength_Seconds float duration of a processing block
ExSynapticStrength float ExSynapticStrength, Ge, given the unit nanosiemens (nS), sets the maximum value of the increment in excitatory conductance associated with an excitatory synaptic input firing. The time course of the conductance increment is a scaled alpha function, t(Ge/Tau)exp(1 - t/Tau), with the time constant of decay, Tau, specified by ExSynapticTau_Seconds.
InhSynapticStrength float InhSynapticStrength, Gi, given the unit nanosiemens (nS), sets the maximum value of the increment in inhibitory conductance associated with an inhibitory synaptic input firing. The time course of the conductance increment is a scaled alpha function, t(Gi/Tau)exp(1 - t/Tau), with the time constant of decay, Tau, specified by InhSynapticTau_Seconds.
SampleRate_Hz float The sample rate, in hertz, sets the effective time resolution of the input and output streams, and the output time resolution of the Runge Kutta differential equation solver.
ExSynapticTau_Seconds float time constant of the excitatory synaptic conductance
InhSynapticTau_Seconds float time constant of the inhibitory synaptic conductance
CellTemperature_DegreesC float The cell temperature (in degrees Celsius) is typically a mammalian body temperature, such as 38, for simulation of in vivo experiments, and 22 for in vitro.
AHPIncrement_Siemens float AHPIncrement_Siemens is an optional parameter, added to the Rothman and Manis 2003c compartment, with a default value of zero. When set to a non-zero value, AHPIncrement_Siemens in combination with TauAHP_Seconds implements adaptation by means of a potassium conductance with a step increase and a first-order exponential decay. Exactly 0.5 ms after each action potential, the conductance is incremented by AHPIncrement_Siemens, and then decays with time constant TauAHP_Seconds.
MaxLag_Seconds float The MaxLag_Seconds parameter sets the maximum characteristic delay in the array of model MSO neurons. See Description for details and alternative usage.
geFilename string An optional parameter, which writes the excitatory synaptic conductance (unit nS) to the specified file.
giFilename string An optional parameter, which writes the inhibitory synaptic conductance (unit nS) to the specified file.
gNaFilename string An optional parameter, which writes the sodium conductance (unit nS) to the specified file.
gKAFilename string An optional parameter, which writes the transient A-type potassium conductance (unit nS) to the specified file.
gKLTFilename string An optional parameter, which writes the low-threshold potassium conductance (unit nS) to the specified file.
gKHTFilename string An optional parameter, which writes the high-threshold potassium conductance (unit nS) to the specified file.
ghFilename string An optional parameter, which writes the hyperpolarization-activated cation conductance (unit nS) to the specified file.
gAHPFilename string An optional parameter, which writes the optional adaptation conductance (unit nS) to the specified file.
TauAHP_Seconds float TauAHP_Seconds is an optional parameter, added to the Rothman and Manis 2003c compartment, with a default value of (1.0/SampleRate_Hz). When parameter AHPIncrement_Siemens is set to a non-zero value, these two parameters implement adaptation by means of a potassium conductance with a step increase and a first-order exponential decay. Exactly 0.5 ms after each action potential, the conductance is incremented by AHPIncrement_Siemens, and then decays with time constant TauAHP_Seconds.
