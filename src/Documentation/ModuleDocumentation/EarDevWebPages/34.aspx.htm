ZhouLSO1.0.0.09/22/2008Anderson DJ
 
Classification:
Structure Substructure Cell
superior olivary complex [Sol], regio olivaris superioris, superior olivary nucleus lateral superior olivary nucleus [LSO], lateral superior olive, accessory superior olive
 
Abstract:
The ZhouLSO module simulates neurons in the lateral superior olive (LSO) that are sensitive to interaural level difference. Each model LSO neuron is an “integrate and fire” electric circuit model with parallel branches containing membrane capacitance and conductances representing leakage, synaptic inputs, refractory behavior, and adaptation (Zhou and Colburn 2008). A voltage source equal to the reversal potential is included in series with each conductance, except leakage. Discharge properties of the model cell suggest that the strength and time constant of adaptation determine four basic response types observed in LSO neurons.
Description:
Each simulation implements a vector array of model LSO neurons, with the vector length equal to the number of frequency channels. Current and synaptic inputs cannot be combined. The module accepts either two synaptic inputs or a single current input. The first synaptic input stream is excitatory, and the second is inhibitory. The excitatory and inhibitory input spike streams can contain multiple inputs, with the number of inhibitory inputs being equal to the number of excitatory inputs.

Parameters:
Name Type Description
SampleRate_Hz float The sample rate, in hertz, sets the effective time resolution of the input and output streams, and the output time resolution of the Runge Kutta differential equation solver.
FrameLength_Seconds float duration of a processing block
C_m_Farads float The membrane capacitance in Farads.
G_L_Siemens float Leakage conductance in Siemens
E_L_Volts float Reversal potential for leakage in Volts
E_abs_Volts float The reversal potenial, in Volts, of the absolute refractory current.
E_AHP_Volts float The reversal potenial, in Volts, of the AHP current.
Tau_AHP_Seconds float The time constant, in seconds, of the AHP conductance.
V_th_Volts float The threshold voltage in Volts. When the membrane reaches the threshold voltage, an action potential is considered to have occurred, and the conductances for absolute refractoriness and adaptation are incremented.
G_e_Siemens float The peak increase in condutance, in Siemens, due to an excitatory input event.
E_e_Volts float The reversal potential for the excitatory synaptic input current.
Tau_e_rise_Seconds float The rise time-constant for the excitatory synaptic conductance. Note that Tau_e_rise_Seconds must be less than Tau_e_fall_Seconds to describe normal causal behavior.
Tau_e_fall_Seconds float The fall time-constant for the excitatory synaptic conductance. Note that Tau_e_rise_Seconds must be less than Tau_e_fall_Seconds to describe normal causal behavior.
G_i_Siemens float The peak increase in condutance, in Siemens, due to an inhibitory input event.
E_i_Volts float The reversal potential, in Volts, for the inhibitory synaptic input current.
Tau_i_rise_Seconds float The rise time-constant for the inhibitory synaptic conductance. Note that Tau_i_rise_Seconds must be less than Tau_i_fall_Seconds to describe normal causal behavior.
Tau_i_fall_Seconds float The fall time-constant for the inhibitory synaptic conductance. Note that Tau_i_rise_Seconds must be less than Tau_i_fall_Seconds to describe normal causal behavior.
VmemFilename string An optional parameter, which writes the membrane voltage in milliVolts to the specified file.
geFilename string An optional parameter, which writes the excitatory synaptic conductance (unit nS) to the specified file.
giFilename string An optional parameter, which writes the inhibitory synaptic conductance (unit nS) to the specified file.
gabsFilename string An optional parameter, which writes the conductance for absolute refractoriness (unit nS) to the specified file.
gAHPFilename string An optional parameter, which writes the adaptation conductance (unit nS) to the specified file.
G_abs_increment_Siemens float The increment in the shunting conductance for absolute refractoriness (expressed in unit Siemens), which occurs when the membrane reaches threshold voltage.
t_abs_seconds float The absolute refractory period in seconds, which begins when the membrane reaches threshold voltage, and during which time the very large absolute refractory condunctance effectively shunts the membrane to its reversal potential (which is typically set to equal the resting membrane potential). At the end the absolute refractory period, the conductance for absolute refractoriness is reset to zero.
G_AHP_increment_Siemens float The increment in conductance for adaptation, in Siemens, which occurs each time the membrane reaches threshold voltage, and causes after-hyperpolarizion (AHP). Unlike absolute refractoriness, gAHP does not reset. gAHP is additive with succesive threshold crossings, and decays exponentially according to its time constant.
V_rest_Volts float The membrane resting potential in Volts, V_rest_Volts, sets the initial membrane voltage in the simulation. Normally the reversal potentials for absolute refractoriness and adaptation are set to the same value as V_rest_Volts, but this convention is not enforced.
