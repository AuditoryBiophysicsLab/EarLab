CaiIC1.0.0.09/18/2008Anderson DJ
 
Classification:
Structure Substructure Cell
inferior colliculus [IC], colliculus caudalis, posterior colliculus central nucleus of inferior colliculus [CIC], Nucleus centralis, chief nucleus of inferior colliculus
 
Abstract:
The CaiIC module implements the Cai et al. (1998) model for neurons of the central nucleus of the inferior colliculus (ICC). While there is wide variation in membrane characteristics between the neurons of this nucleus, and across their population they receive inputs from a number of sources, the Cai et al. IC model focuses on the subpopulation of ICC neurons that are sensitive to interaural time delay (ITD) and respond best to low-frequency auditory inputs. Each instance of the CaiIC module incorporates a single-compartment Hodgkin-Huxley neuron model defined by an electric circuit equation incorporating membrane currents associated with capacitance, leakage, ion channels, and synaptic inputs. The characteristics of the three voltage-sensitive ion-channel conductances (inactivating sodium gNa, delayed-rectifier high-threshold potassium gKHT, and low-threshold potassium gKLT) were derived from voltage-clamp data taken from the ventral cochlear nucleus (VCN) (Rothman et al. 1993; Manis and Marx, 1991). The Cai98 compartment incorporates an additional conductance for additive long-lasting adaptation following each action potential, a characteristic of certain ICC neurons. With the exception of this added conductance, the neural compartment is that of Brughera et al. (1996), including its modifications from the Rothman et al. (1993) compartment, which smooth and slow the ion-channel activity slightly and increase the overall potassium conductance near resting potential. The module’s ResponseType parameter can be used to select Type I or II responses. The default response is Type II (phasic), characteristic of cells that are effective in phase-locking and entraining to low frequency auditory stimuli. Selecting a Type I (sustained) response eliminates gKLT, and modifies the strengths of capacitance, gNa, and gKHT. Although adaption characteristics and response type are selected through independent parameters, in IC neurons the K+ current that supports adaptation is co-expressed with the basic ion-channel currents that support a Type I response, but not with the additional currents associated with a Type II response (Sivaramakrishnan and Oliver, 2001).
Description:
The CaiIC module implements an array of ICC neurons with two dimensions of lengths that are specified in the runtime gui or diagram file. The length of the first dimension sets the number of frequency channels, and the length of the second dimension sets the number of model cells per frequency channel equal to the number of characteristic delays in each of two MSO modules acting as an input. In order to implement the Cai et al. model (1998), the CaiIC module receives one excitatory input stream driven by an ipsilateral MSO module, and one inhibitory input stream driven by a contralateral MSO module. Each model ICC neuron receives one excitatory synaptic input, and one inhibitory synaptic input. While this strict adherence to the original model may be restrictive, the CaiIC module thus incorporates the original model’s convergent inputs of same and different characteristic delays. The inputs, individual cells within the BrugheraMSO module, are organized by frequency channel and characteristic delay. The number of frequency channels is equal between ipsilateral and contralateral MSO modules, as are the number of characteristic delays. Within a frequency channel in the MSO modules, the characteristic delays progress from zero to the maximum value in equal increments. The CaiIC module has the same number of frequency channels as each MSO module. Within a frequency channel the CaiIC module has a number of neurons equal to the number of characteristic delays in each MSO module, and within that frequency channel each contralateral model MSO cell (with its different characteristic delay) projects an inhibitory input to a different model IC neuron. Setting the ExcitatoryMSOLagNumber parameter (to a value between zero and the number of characteristic delays minus one) selects exactly one ipsilateral model MSO cell of that particular characteristic delay so that it projects an excitatory input to each of the model IC neurons within that frequency channel. In this way the effects of similarity and differences in the characteristic delays between the excitatory input and inhibitory input can be examined.

Parameters:
Name Type Description
SampleRate_Hz float The sample rate, in hertz, sets the effective time resolution of the input and output streams, and the output time resolution of the Runge Kutta differential equation solver.
FrameLength_Seconds float duration of a processing block
ExSynapticStrength float ExSynapticStrength, Ge, given the unit nanosiemens (nS), sets the maximum value of the increment in excitatory conductance associated with an excitatory synaptic input firing. The time course of the conductance increment is a scaled alpha function, t(Ge/Tau)exp(1 - t/Tau), with the time constant of decay, Tau, specified by ExSynapticTau_Seconds.
ExSynapticTau_Seconds float time constant of the excitatory synaptic conductance
InhSynapticStrength float InhSynapticStrength, Gi, given the unit nanosiemens (nS), sets the maximum value of the increment in inhibitory conductance associated with an inhibitory synaptic input firing. The time course of the conductance increment is a scaled alpha function, t(Gi/Tau)exp(1 - t/Tau), with the time constant of decay, Tau, specified by InhSynapticTau_Seconds.
InhSynapticTau_Seconds float time constant of the inhibitory synaptic conductance
CellTemperature_DegreesC float The cell temperature (in degrees Celsius) is typically a mammalian body temperature, such as 38, for simulation of in vivo experiments, and 22 for in vitro.
AHPIncrement_Siemens float AHPIncrement_Siemens is an optional parameter, added to the Rothman and Manis 2003c compartment, with a default value of zero. When set to a non-zero value, AHPIncrement_Siemens in combination with TauAHP_Seconds implements adaptation by means of a potassium conductance with a step increase and a first-order exponential decay. Exactly 0.5 ms after each action potential, the conductance is incremented by AHPIncrement_Siemens, and then decays with time constant TauAHP_Seconds.
InhInputDelay_Seconds float The delay in seconds applied to the inhibitory input.
ExcitatoryMSOLagNumber int Setting the ExcitatoryMSOLagNumber parameter (to a value between zero and the number of characteristic delays minus one) selects exactly one ipsilateral model MSO cell of that particular characteristic delay so that it projects an excitatory input to each of the model IC neurons within that frequency channel.
ResponseType int An optional parameter, with a default value of 2 (Type II). Type I: ResponseType = 1; Type II: ResponseType = 2.
geFilename string An optional parameter, which writes the excitatory synaptic conductance (unit nS) to the specified file.
giFilename string An optional parameter, which writes the inhibitory synaptic conductance (unit nS) to the specified file.
gNaFilename string An optional parameter, which writes the sodium conductance (unit nS) to the specified file.
gKHTFilename string An optional parameter, which writes the high-threshold potassium conductance (unit nS) to the specified file.
gKLTFilename string An optional parameter, which writes the low-threshold potassium conductance (unit nS) to the specified file.
gAHPFilename bool An optional parameter, which writes the optional adaptation conductance (unit nS) to the specified file.
TauAHP_Seconds float TauAHP_Seconds is an optional parameter, added to the Rothman and Manis 2003c compartment, with a default value of (1.0/SampleRate_Hz). When parameter AHPIncrement_Siemens is set to a non-zero value, these two parameters implement adaptation by means of a potassium conductance with a step increase and a first-order exponential decay. Exactly 0.5 ms after each action potential, the conductance is incremented by AHPIncrement_Siemens, and then decays with time constant TauAHP_Seconds.
